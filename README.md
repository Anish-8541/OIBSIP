# 🌡 Temperature Converter

A responsive Temperature Converter built using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development & Designing Internship.

🚀 Features

- Celsius to Fahrenheit Conversion
- Fahrenheit to Celsius Conversion
- Kelvin Support
- Conversion History
- Copy Result Functionality
- Dark/Light Theme Toggle
- Responsive User Interface

🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
